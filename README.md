FastPrep DSA App

Overview
.FastPrep DSA App is a full-stack web application designed to help users prepare for Data Structures and Algorithms efficiently. It provides problem tracking, authentication, and a clean user interface for a smooth learning experience.

Features
.User Authentication (Login/Signup)
.Track DSA problems and progress
.Clean and responsive UI
.Full-stack architecture (frontend + backend)

Tech Stack
.Frontend: React.js, HTML, CSS
.Backend: Node.js, Express.js
.Database: MongoDB
.Tools: Git, REST APIs

Project Structure
.backend/   → API, routes, database  
.frontend/  → React UI
